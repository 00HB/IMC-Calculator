# Calculadora de IMC en Python

## Descripción

Este proyecto es un programa en Python que solicita datos personales del usuario y calcula su Índice de Masa Corporal (IMC).

El programa pide nombre, apellido paterno, apellido materno, edad, peso y estatura. Después muestra los datos capturados junto con el IMC calculado.

## Objetivo

Crear un programa que pida al usuario su nombre, apellido paterno, apellido materno, edad, peso y estatura, y que despliegue esta información en pantalla junto con su Índice de Masa Corporal.

## Funcionalidades

- Captura de datos desde teclado.
- Validación de campos vacíos.
- Validación de datos numéricos.
- Validación de valores mayores que cero.
- Cálculo del IMC.
- Impresión de los datos capturados y del resultado final.

## Requisitos

- Python 3

## Cómo ejecutar el programa

1. Clonar este repositorio o descargar el archivo del proyecto.
2. Abrir una terminal en la carpeta del proyecto.
3. Ejecutar el programa con el siguiente comando:

python calculadora_imc.py

En algunos sistemas puede ser necesario usar:

python3 calculadora_imc.py

## Fórmula utilizada

IMC = peso / estatura²

## Ejemplo de uso

Ingresa tu nombre: Pablo
Ingresa tu apellido paterno: Márquez
Ingresa tu apellido materno: Gutiérrez 
Ingresa tu edad: 25
Ingresa tu peso en kilogramos: 70
Ingresa tu estatura en metros, por ejemplo 1.70: 1.75

--- DATOS CAPTURADOS ---
Nombre: Pablo
Apellido paterno: Márquez
Apellido materno: Gutiérrez
Edad: 25 años
Peso: 70.0 kg
Estatura: 1.75 m

--- RESULTADO ---
IMC calculado: 22.86
